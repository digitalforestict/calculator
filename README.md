# calculator
complete, modern calculator project with HTML, CSS, and JavaScript all in one

Key Features of This Calculator:
1. Complete Calculator Functionality
Basic Operations: Addition, subtraction, multiplication, division

Advanced Features: Percentage calculations, decimal support

Clear Functions: Clear all (C), delete last digit (backspace)

Equals: Performs calculations with proper order of operations

2. Modern UI/UX Design
Clean, modern interface with gradient backgrounds

Responsive design that works on all devices

Visual feedback with hover effects and animations

Professional color scheme with indigo primary color

3. Interactive Features
Dark/Light Theme Toggle: Switch between themes with persistent storage

Calculation History: Stores previous calculations with timestamps

Click History Items: Reuse previous results by clicking on them

Keyboard Support: Full keyboard navigation (numbers, operators, Enter, Escape, Backspace)

4. Technical Implementation
Local Storage: Saves theme preference and calculation history

Error Handling: Division by zero protection

Precision Control: Limits decimal places to avoid floating point issues

Animation: Visual feedback when calculations occur

6. User Experience Features
Live Display: Shows current input and previous calculations

History Panel: Scrollable list of past calculations

Visual Feedback: Buttons animate on click

How to Use:
Basic Calculations:

Click numbers (0-9) to input values

Click operators (+, −, ×, ÷) to select operation

Click equals (=) to get result

Special Functions:

C: Clear all (reset calculator)

←: Delete last digit

%: Convert to percentage

.: Add decimal point

History Features:

View past calculations in the history panel

Click any history item to reuse the result

Clear all history with the "Clear History" button

Theme:

Toggle between dark and light mode using the theme button

Preference is saved automatically

Keyboard Shortcuts:

Numbers: 0-9

Operators: +, -, *, /, %

Equals: = or Enter

Clear: Escape

Delete: Backspace

This calculator is completely self-contained in a single HTML file and can be deployed directly to GitHub Pages or any web server without any additional dependencies
